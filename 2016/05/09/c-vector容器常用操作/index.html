c++ vector容器常用操作

  • c++
  • vector

其实这东西从来都是屯在我的EverNote里的，今天又用到了，还是整理出来吧。。

Note：

  1. 尽量使用reserve来减少不必要的内存分配次数。

  2. 原则：尽量使用empty()而不是size()==0 来判断容器是否为空

0x00 vector 的初始化

vector<int> v2(10); //初始化size为10可以避免数组动态增长的时候不断的分配内存
//或者vec.reserve(10); 功能同上

int  v1[10] = {0,1,0,0,3,0,0,4,4,4};
vector<int> v3(&v1[0],&v1[9]);//原始数组的元素指针可以作为迭代器来使用

0x01. 使用迭代器访问元素

vector<int>::iterator it;

for(it=vec.begin();it!=vec.end();it++)

cout<<*it<<endl;

0x02 和其他容器都类似，支持begin和end获取首尾指针

//传入sort函数
sort(vec.begin(), vec.end(), Cmp);

0x03 获取首尾元素和大小情况

vec.front(); vec.back(); vec.size()； vec.empty();

0x04 元素的插入和删除

//在尾部增加和删除元素

vec.push_back();

vec.pop_back(); //可用来模仿stack和quene

vec.clear() ; //清除元素

vec.insert(vec.begin()+i,a);在第i+1个元素前面插入a;

// erase the 6th element
myvector.erase (myvector.begin()+5);

// erase the first 3 elements:
myvector.erase (myvector.begin(),myvector.begin()+3); //其实一般都是左闭右开 <=  <

0x05 常用算法

//1. sort

sort(vec.begin(),vec.end());(默认是按升序排列,即从小到大).

//可以通过重写排序比较函数按照降序比较，如下：

//定义排序比较函数：

bool Comp(const int &a,const int &b)
{
    return a>b;
}
//调用时:
sort(vec.begin(),vec.end(),Comp)，这样就降序排序。其他类型的一样，    总之最后返回bool值就行

//2. reverse
reverse(vec.begin(),vec.end());将元素翻转
